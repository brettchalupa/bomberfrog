# Bomberfrog

**In Development**

Horizontal wave-based shmup by Brett Chalupa made with
[Usagi Engine](https://usagiengine).

## Developing

Run `usagi dev` to boot up the dev server.

Push the latest build to itch with `./push.rb`.

Dev mode shortcuts:

- <kbd>1</kbd> - toggle player invincibility
